# PT Tunas Tumbur Sejahtera – Company Website

Welcome to the official website repository of **PT Tunas Tumbur Sejahtera (TTS Transport)**, a logistics and trucking company that provides reliable transportation services across Indonesia.

This website is designed with a clean, modern interface, responsive layout, and clear navigation to showcase the company’s services, fleet, and project portfolio.

---

## What Makes This Website Special?

### Complete & Professional Fleet Showcase

The website presents a full list of available fleet types, including Blindvan, Pickup, CDE, CDD, Fuso, Wingbox, Tronton, and Trailer 20ft/40ft. All images and descriptions are organized neatly for easy viewing.

### Multi-Language Support

Visitors can switch between Indonesian and English.
The site automatically saves language preferences for future visits.

### Dynamic Portfolio Gallery

The portfolio section displays shipment and project images through:

* Category filters
* Grid layout
* Modal zoom
* Smooth hover animations
* Lazy-loading for better performance

### Fully Responsive Layout

The entire website adapts to desktop, tablet, and mobile views.
Mobile users get a streamlined menu with a sliding navigation drawer.

### Clean UI and Soft Animations

The design incorporates:

* Glass-style navigation bar
* Subtle hero animations
* Fade-in effects on scroll
* Consistent brand color palette

## How to Run the Project

This project is fully static and requires no additional setup.

1. Clone the repository:

   ```bash
   git clone https://github.com/USERNAME/REPO-NAME.git
   ```

2. Open the folder:

   ```bash
   cd REPO-NAME
   ```

3. Launch the main file:

   ```
   index.html
   ```

4. Open it in any modern browser.

## Project Folder Structure

```
/asset
   /portofolio        → Portfolio images
   background.jpg     → About section background

/data
   portofolio.json    → Optional portfolio data source

index.html            → Main page
portfolio.html        → Portfolio page
styles.css            → Main stylesheet
mobile.css            → Mobile-specific styles
script.js             → Main JavaScript
mobile.js             → Mobile navigation script
portofolio.js         → Portfolio page script
```

## Deployment Options

### 1. GitHub Pages

* Go to Settings → Pages
* Select the `main` branch
* Save and publish

### 2. Shared Hosting / cPanel

Upload all files into the `public_html` directory.

### 3. Netlify or Vercel

Drag and drop the project folder to publish instantly.
